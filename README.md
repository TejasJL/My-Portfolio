# My Digital Resume (Engineering Portfolio)

A sleek, responsive, and minimalist interactive digital resume built to showcase my engineering journey, technical stack, and software projects. 

🚀 **Live Deployment:** [https://main.d4u3m40j24yvi.amplifyapp.com/](https://main.d4u3m40j24yvi.amplifyapp.com/)

---

## 🛠️ Technical Stack

- **Frontend:** React.js, Tailwind CSS, Framer Motion (for smooth animations), Lucide React (Icons)
- **Tooling & Bundler:** Vite
- **Cloud & Deployment:** AWS Amplify (CI/CD pipeline integrated with GitHub)

---

## ⚡ Key Features

- **Interactive Terminal Hero:** A humanoid terminal experience introducing my technical domain and career objectives.
- **Production-Ready Cloud Hosting:** Fully deployed on AWS infrastructure utilizing standard cloud best practices.
- **Fully Responsive:** Optimized for flawless viewing across mobile devices, tablets, and wide monitors.
- **Clean Architecture:** Component-driven structure ensuring high performance, quick build times, and maintainability.

---

## 🚀 Cloud Infrastructure & Deployment

This project is hosted on **AWS Amplify**, establishing a modern DevOps workflow:
1. **Continuous Integration (CI):** Any changes pushed to the `main` branch automatically trigger a secure cloud build pipeline.
2. **Global Delivery:** Hosted using AWS's global Content Delivery Network (CDN) ensuring ultra-fast loading times and high availability.
3. **Security:** Served entirely over secure HTTPS using managed SSL/TLS certificates.
